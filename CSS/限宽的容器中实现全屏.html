      # 限宽的容器中实现全屏

      • 在Web布局中，特别是在PC端中，常常可以看到这样的设计风格：
      • 内容居中，然后Banner区全屏。这也就是标题所说的效果，限宽的容器中实现全屏效果。

      # 问题

      • 在实际中如何实现在限宽的容器中实现全屏效果。
      • 那么针对下面的结构，如何让.full-width中的img能实现全屏的效果

    <div class="container"> 
         <p>....</p> 
         <figure class="full-width"> 
              <img src="" alt="">

      # (2) 视窗单位vw

      视窗，指的是浏览器屏幕。1vw就意味着1%的浏览器的宽度。100vw将意味着整个浏览器宽度。

      只需要设置其width值为100vw就能拿到浏览器全屏的宽度 把.full-width容器向右推50%，然后使用margin-left: -50vw拉回去。以同样的原理，把容器向左推50%，并使用margin-right: -50vw向右拉出去。容器虽然其父容器有宽度限制水平居中，但经过拉伸推移，就实现了容器的全屏效果

      .container {
  max-width: 960px;
  margin: 0 auto;
}
.full-width {
  min-height: 200px;
  background: rgba(255, 165, 0, 0.5);
  width: 100vw;
  position: relative;
  left: 50%;
  right: 50%;
  margin-left: -50vw;
  margin-right: -50vw;
}

      # （3）使用translateX & calc()

      容器的宽度是960px，全屏的宽度是100vw，他们之间相差就是960px - 100vw， 但我们只需要向左拉出其值的一半，也就是(960px - 100vw) / 2。

      .full-width {
  min-height: 200px;
  background: rgba(255, 165, 0, 0.5);
  width: 100vw;
  transform: translateX(calc((960px - 100vw) / 2));
}

      # 优化

      960px相当于100%，这样一来960px / 2也就相当于50%；对应的100vw / 2就变成了50vw。

      .full-width {
  min-height: 200px;
  background: rgba(255, 165, 0, 0.5);
  margin-left: calc(50% - 50vw);
  margin-right: calc(50% - 50vw);
}

      # Sass改造 Sass的混合宏对其进行封装

      (1) $support-type，可以给他传递三个值： margin：对应CSS中margin-left和margin-right的方案 position: 对应CSS中position定位的方案 translate: 对应CSS中translateX的方案

      (2) $min-width: 容器宽度值

      @mixin full-width($support-type:margin, $min-width:null) {
    @if $support-type == 'margin' {
        margin-left: calc(-50vw + 50%);
        margin-right: calc(-50vw + 50%);
        // margin-left: calc(-100vw / 2 + #{$min-width} / 2);
        // margin-right: calc(-100vw / 2 + #{$min-width} / 2);
    }

    @if $support-type == 'position' {
        width: 100vw;
        position: relative;
        left: 50%;
        right: 50%;
        margin-left: -50vw;
        margin-right: -50vw;
    }

    @if $support-type == 'translate' {
        width: 100vw;
        transform: translateX(calc((#{$min-width} - 100vw) / 2));
    }
}
.full-width {
    @include full-width(translate, 960px);
}
